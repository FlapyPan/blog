# HTML 防蜘蛛

> 都是防君子不防小人的方法

使用 HTML 标签，告诉搜索引擎不需要索引

```html
<meta name="robots" content="noindex, nofollow" />
```

使用 robots.txt

```text
User-agent: *
Disallow: *
```
